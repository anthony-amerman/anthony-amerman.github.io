what are the best practices associated with using classes vs. ids?

ids

an id is a unique attribute assigned to an html element in its opening tag using: id="idnamehere". it cannot be assigned to any other elements. it can then be used in CSS as a selector in order to style that specific element using: #idnamehere{ }. here is what id looks like in both html and css:

classes

a class is an attribute assigned to an html element in its opening tag using class="classnamehere". unlike id it is not unique, meaning it can be assigned to multiple elements and elements can be assigned multiple classes. this can then be used in css as a selector in order to style every element belonging to that class using: .classnamehere{ } here is what class looks like in both html and css:

summary

both id and class are used to style elements in css. id is for giving a specific element a unique id, while class is used to sort elemnts into groups, to apply the same styling over an assortment of different elements.
